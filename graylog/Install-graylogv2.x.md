####Cách cài đặt
 - Cấu hình tối thiểu yêu : RAM 2GB, CPU = 2 Core.
 
 - Với Graylog Server
 
 ```sh
 wget https://raw.githubusercontent.com/hocchudong/ghichep-graylog/master/graylog/graylog-scripts/graylog2-0.sh
 ```
 - Với Client : Thực hiện theo link sau.
